# Constitution Enforcement Rules

1. All code must originate from specs.
2. No skill may modify files outside its domain.
3. Subagents must use templates.
4. All modifications to existing files require user confirmation.
5. Specs must not be ignored.
6. Constitution cannot be changed programmatically.
7. System files (node_modules, os paths) are forbidden.
8. Backend and frontend must remain separated.
