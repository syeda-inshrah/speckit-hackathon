# TailwindCSS Component Guidelines

## General
- Use flex, grid, gap, rounded, p-*, text-*
- Avoid custom CSS
- Use clsx if class merging is needed

## Spacing
- Default padding: p-4
- Default gap: gap-4

## Layout
- Use flex-col for vertical components
- Use text-sm or text-base for general text
