# UI Patterns Reference

## Page Structure
Use semantic HTML:
<header>, <section>, <main>, <footer>

## Empty State Pattern
If no tasks:
<p className="text-gray-500">No tasks available</p>

## Error State
<p className="text-red-600">Error loading data.</p>

## Button Pattern
<button className="px-4 py-2 bg-blue-600 text-white rounded">
